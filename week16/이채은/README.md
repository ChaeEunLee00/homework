# ▶숙제

### BOJ 15787 기차가 어둠을 헤치고 은하수를
- 구현 + 비트마스킹
  - 비트마스킹: 비트를 활용해 데이터를 효율적으로 저장하고 조작하는 것
    - `&`: AND 연산
    - `|`: OR 연산
    - `^`: XOR 연산
    - `~`: 모든 비트 반전
    - `<<`: 모든 비트 왼쪽 이동
    - `>>`: 모든 비트 오른쪽 이동

### BOJ 2064 IP 주소
- 구현 + 비트마스킹
  - 네트워크 마스크: 모든 ip에 대해 같은 위치의 비트가 같은지 비교 -> 같으면 1, 다르면 그 이후부터는 모두 0
  - 네트워크 주소: 한 ip 주소와 네크워크 마스크의 교집합
